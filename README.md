# Gmail 領収書自動保存システム (Gmail Receipt to Drive)

Gmail上の特定のラベルがついたメールから領収書PDFを抽出し、Googleドライブの月別フォルダへ自動整列させるシステムです。

## 1. システム概要

本システムは、個人事業主や法人の経理作業を効率化するために、以下の処理を自動化します。

- **抽出**: 領収書振り分けラベルが付与されたメールをスキャン
- **判定**: 添付ファイル（PDF）の有無を確認。Receipt（領収書）を最優先で取得
- **PDF化**: 添付ファイルがない場合、メール本文をPDFとして出力
- **整理**: Googleドライブ内に「年/月」のフォルダ構造を自動生成し、`YYYYMMDD_企業名.pdf` の形式で保存
- **管理**: 処理済みスレッドに領収書ラベルを付与し、領収書振り分けラベルを削除

## 2. フォルダ構造イメージ

```
[Google Drive Root]
  └── 2026 (Year Folder)
      ├── 01 (Month Folder)
      ├── 02
      └── 03
          ├── 20260315_Amazon_co_jp.pdf
          ├── 20260320_Google_Cloud_本文.pdf
          └── 20260322_Urawa_Reds_Official.pdf
```

## 3. セットアップガイド

### 3.1 前準備

1. Googleドライブに「親フォルダ」を作成し、ブラウザのURLからフォルダIDをメモします
2. Gmailで `領収書振り分け` という名前のラベルを作成します
3. GASエディタの「プロジェクトの設定 → スクリプトプロパティ」に以下を登録します

| プロパティ名 | 値 | 必須 |
|---|---|---|
| `PARENT_FOLDER_ID` | GoogleドライブのフォルダID | Yes |
| `TARGET_LABEL` | 対象ラベル名（デフォルト: `領収書振り分け`） | No |
| `DONE_LABEL` | 処理済みラベル名（デフォルト: `領収書`） | No |

### 3.2 開発環境（clasp使用時）

1. `clasp login` で認証
2. `clasp clone [ScriptID]` でプロジェクトを同期
3. `src/main.ts`（または `main.js`）にコードを配置

## 4. 自動化の設定

1. GASエディタの「トリガー（時計アイコン）」を開く
2. 「トリガーを追加」をクリック
3. 「イベントのソース」を「時間主導型」に設定
4. 「日付ベースのタイマー」を選択し、任意の時刻（例：午前2時〜3時）に設定して保存

## 5. 技術仕様・制限事項

| 項目 | 説明 |
|---|---|
| **二重処理防止** | スレッド単位でターゲットラベルを剥がすため、次回の実行対象から自動的に外れる |
| **API権限** | 実行時に Gmail, Drive, Document へのアクセス権限が必要 |
| **クォータ制限** | Googleの実行時間制限（6分/回）あり。初回は少なめのメール数でテスト推奨 |
